# 解答
Calendarクラスのインスタンスに対して、`set(Date)`メソッドで現在時刻のDateクラスのインスタンスを渡す。  
このCalendarクラスのインスタンスは、現在時刻を示しているが、これに対して時、分、秒、ミリ秒をランダムな値で加算することで、現在時刻からプラスマイナス1日の範囲でランダムな日時を作成する。
